# 更新日誌

## v0.1.1 (優化版本)

### 🔒 安全性改進

1. **多重任務根目錄保護**
   - 在掃描遞迴入口處檢查，防止掃描任務根目錄
   - 在路徑優化階段檢查，防止將任務根目錄加入刪除列表
   - 在執行刪除階段檢查，最後一道防線
   - 添加路徑範圍驗證，確保刪除的文件在任務資料夾內

2. **Target 匹配邏輯修正**
   - 修正了有 target 配置時的匹配邏輯
   - 確保只有匹配 target 的項目才會被刪除
   - 正確處理資料夾繼承匹配狀態

3. **配置驗證增強**
   - 驗證資料夾路徑是否存在
   - 驗證 glob 模式是否有效
   - 警告過短的掃描間隔（防止資源浪費）
   - 警告過短的閾值（防止誤刪）

### 🛠️ 穩定性改進

1. **配置熱重載優化**
   - 只有成功載入新配置才更新
   - 載入失敗時保持原有配置繼續運行
   - 避免因配置錯誤導致程序停止工作

2. **內存管理優化**
   - 優化數據庫清理邏輯，批量處理減少內存分配
   - 自動清理不存在的資料夾記錄
   - 使用 filter_map 減少中間集合

3. **錯誤處理改進**
   - 添加刪除統計信息
   - 更詳細的錯誤日誌
   - 跳過不存在的文件而不是報錯

### 📊 可維護性改進

1. **代碼結構優化**
   - 更清晰的註釋和文檔
   - 統一的錯誤處理模式
   - 更好的函數職責劃分

2. **日誌改進**
   - 添加刪除操作統計
   - 更清晰的警告和錯誤信息
   - 區分不同級別的安全檢查

### 📚 文檔改進

1. **完整的 README.md**
   - 詳細的功能說明
   - 多個使用場景示例
   - Glob 模式詳細說明
   - 常見問題解答
   - 安全機制說明

2. **配置文件示例**
   - config.example.toml：完整的配置示例
   - config.test.toml：測試配置模板
   - 詳細的註釋說明

### 🐛 Bug 修復

1. 修復任務根目錄可能被刪除的問題
2. 修復 target 配置不生效的問題
3. 修復配置重載時計時器不重置的問題
4. 修復數據庫清理可能遺漏空資料夾的問題

## v0.1.0 (初始版本)

### ✨ 功能特性

- 智能掃描：支持快速掃描和完整掃描
- 精確匹配：使用 glob 模式匹配目標文件
- 安全刪除：支持移入垃圾桶或徹底刪除
- 數據持久化：使用高效的二進制數據庫
- 系統托盤：最小化到系統托盤
- 配置熱重載：運行時刷新配置
- 詳細日誌：完整的操作記錄

### 🏗️ 技術架構

- Rust 語言開發
- rkyv 高效序列化
- globset 模式匹配
- winit 事件循環
- tray-icon 系統托盤
